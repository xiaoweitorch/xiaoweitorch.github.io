Div Game

编号：0024

试题来源：AtCoder

题目描述

给定一个正整数\(N\)，思考对\(N\)进行下面重复的操作

  • 首先，选定一个正整数\(z\)满足下面所有的要求：
    • \(z\)能够被表示为\(z=p^e\)，这里面的\(p\)是质数，\(e\)是正整数
    • \(N\%z ==0\)
    • \(z\)和先前选择的整数不同
  • 然后继续对\(N/z\)进行上述操作

找到这个操作能够进行的最大次数

条件

  • 所有的输入都是整数
  • \(1\leq N\leq 10^{12}\)

输入

N

解答算法

算法思路

题目中要求\(z = p^e\)，而且\(p\)又是质数，然后\(z\)还要求是原来数字\(N\)的一个因数，很显然这道题目相当于质因数分解的题目。

假设我们已经解决了质因数分解的问题，已经得到了数字\(N\)分解出的质因数，那么我们会有一个map，对于每个数字\(N\)的质因数，对应的值是其出现的次数，例如\(8\)的质因数为(<2,3>)，表明2作为其质因数，出现了3次。

要想弄清楚操作的次数，对于任意的一个质因数\(<i,k>\)，最开始，可以选取\(i^1\)作为\(z\)，然后可以选取\(i^2\)，依次增大，一直到\(1+2+\dots +j\leq k 且1+2+\dots+j+(j+1)>k\)，那么\(j\)就是对于质因数\(<i,k>\)所能进行的最多操作次数。要求整体的最多操作次数，只要求\(\sum\limits_{map.begin()}^{map.size}j\)。

因此这道题目的关键在于求解质因数。

求解质因数可以采用递归的思路进行处理。首先设置工具函数is_prime来检验数\(i\)是否为质数。然后对于一个数\(n\)，求解其质因数，从2开始进行遍历，遍历到\(n\)为止，判断其中是否存在一个数\(i\)是质数，同时满足\(n\%i==0\)。找到这样的数，循环终止。对\(n/i\)调用上述过程。如果\(n/i\)是质数，递归终止。

复杂度分析

  • 时间复杂度：\(O(n^2)\)
  • 空间复杂度：\(O(n)\)
